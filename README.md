# trade-desk

The Trade Desk — independent, omnichannel demand-side platform (DSP) for programmatic advertising. Steward of Unified ID 2.0 (UID2), OpenPath, Koa AI, Galileo, and OpenSincera. Profiles the public Workflows API, Data API, Platform API, Real-Time Conversion Events API, UID2, and OpenSincera surfaces.
